# Rencana Penelitian: Integrasi Telegram Bot API

## Tujuan Penelitian

1. Pembuatan dan setup bot Telegram
2. Endpoint API untuk pengiriman pesan
3. Autentikasi dan manajemen token
4. Format pesan untuk alert
5. Setup webhook untuk notifikasi real-time

# Rencana Penelitian: Polymarket API

## Tujuan Penelitian
Meneliti Polymarket API secara komprehensif untuk memahami:
1. Cara mendapatkan daftar pasar/event (terutama crypto dan political events)
2. Endpoint API untuk data pasar, odds/harga
3. Rate limits dan kebutuhan autentikasi
4. Struktur dan format data
5. Kemampuan akses data real-time
6. Endpoint data historis yang tersedia
